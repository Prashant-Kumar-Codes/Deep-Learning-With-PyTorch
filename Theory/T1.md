**<span style="font-size: 33px;">PyTorch Basics Thoery + Code Summary
</span>**
---------

* **<span style="font-size: 20px;">Definition:</span>**<span style="font-size: 20px;"> An open-source framework for building and training neural networks using **Tensors** and **Dynamic Computation Graphs**.</span>
* **<span style="font-size: 20px;">Key Engine:</span>**<span style="font-size: 20px;"> **Autograd**, which handles automatic differentiation for backpropagation.</span>
* **<span style="font-size: 20px;">Computational Advantage:</span>**<span style="font-size: 20px;"> Tensors are optimized for **GPU acceleration** and follow the **Define-by-Run** philosophy (graphs are built during execution).</span>

***

## **<span style="font-size: 20px;">1. Tensor Creation Functions</span>**

<span style="font-size: 20px;">Used to initialize data containers with specific distributions or properties.</span>

| <span style="font-size: 20px;">Function</span>         | <span style="font-size: 20px;">Definition</span>                                                                    | <span style="font-size: 20px;">Syntax</span>                                        |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| **<span style="font-size: 20px;">`tensor()`</span>**   | <span style="font-size: 20px;">Creates a tensor from existing data (copies data).</span>                            | <span style="font-size: 20px;">`torch.tensor(data, dtype=None, device=None)`</span> |
| **<span style="font-size: 20px;">`zeros()`</span>**    | <span style="font-size: 20px;">Creates a tensor filled with 0s.</span>                                              | <span style="font-size: 20px;">`torch.zeros(size)`</span>                           |
| **<span style="font-size: 20px;">`ones()`</span>**     | <span style="font-size: 20px;">Creates a tensor filled with 1s.</span>                                              | <span style="font-size: 20px;">`torch.ones(size)`</span>                            |
| **<span style="font-size: 20px;">`rand()`</span>**     | <span style="font-size: 20px;">Uniform random values between</span> $[0, 1)$<span style="font-size: 20px;">.</span> | <span style="font-size: 20px;">`torch.rand(size)`</span>                            |
| **<span style="font-size: 20px;">`randn()`</span>**    | <span style="font-size: 20px;">Normal distribution (mean=0, std=1).</span>                                          | <span style="font-size: 20px;">`torch.randn(size)`</span>                           |
| **<span style="font-size: 20px;">`randint()`</span>**  | <span style="font-size: 20px;">Random integers in a specific range.</span>                                          | <span style="font-size: 20px;">`torch.randint(low, high, size)`</span>              |
| **<span style="font-size: 20px;">`arange()`</span>**   | <span style="font-size: 20px;">Returns a 1D tensor with a sequence of steps.</span>                                 | <span style="font-size: 20px;">`torch.arange(start, end, step)`</span>              |
| **<span style="font-size: 20px;">`linspace()`</span>** | <span style="font-size: 20px;">Evenly spaced values over a specified interval.</span>                               | <span style="font-size: 20px;">`torch.linspace(start, end, steps)`</span>           |
| **<span style="font-size: 20px;">`eye()`</span>**      | <span style="font-size: 20px;">Creates an identity matrix (diagonal 1s).</span>                                     | <span style="font-size: 20px;">`torch.eye(n)`</span>                                |
| **<span style="font-size: 20px;">`*_like()`</span>**   | <span style="font-size: 20px;">Matches properties of an existing tensor.</span>                                     | <span style="font-size: 20px;">`torch.zeros_like(input_tensor)`</span>              |

## **<span style="font-size: 20px;">2. Tensor Operations</span>**

<span style="font-size: 20px;">The mathematical building blocks for manipulating data.</span>

| <span style="font-size: 20px;">Function/Operator</span>    | <span style="font-size: 20px;">Definition</span>                                       | <span style="font-size: 20px;">Syntax</span>                             |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| **<span style="font-size: 20px;">`add()` / `+`</span>**    | <span style="font-size: 20px;">Element-wise addition; supports `alpha` scaling.</span> | <span style="font-size: 20px;">`torch.add(input, other, alpha=1)`</span> |
| **<span style="font-size: 20px;">`mul()` / `*`</span>**    | <span style="font-size: 20px;">Element-wise multiplication.</span>                     | <span style="font-size: 20px;">`torch.mul(input, other)`</span>          |
| **<span style="font-size: 20px;">`matmul()` / `@`</span>** | <span style="font-size: 20px;">Matrix multiplication (Linear Algebra rules).</span>    | <span style="font-size: 20px;">`torch.matmul(input, other)`</span>       |
| **<span style="font-size: 20px;">`to()`</span>**           | <span style="font-size: 20px;">Casts tensor to a new `dtype` or `device`.</span>       | <span style="font-size: 20px;">`tensor.to(torch.float32)`</span>         |
| **<span style="font-size: 20px;">`view()`</span>**         | <span style="font-size: 20px;">Reshapes a tensor (shares underlying memory).</span>    | <span style="font-size: 20px;">`tensor.view(shape)`</span>               |
| **<span style="font-size: 20px;">`reshape()`</span>**      | <span style="font-size: 20px;">Changes shape (may copy data if needed).</span>         | <span style="font-size: 20px;">`tensor.reshape(shape)`</span>            |
| **<span style="font-size: 20px;">`equal()`</span>**        | <span style="font-size: 20px;">Checks if two tensors have identical data/shape.</span> | <span style="font-size: 20px;">`torch.equal(tensor1, tensor2)`</span>    |

## **<span style="font-size: 20px;">3. Aggregation Operations</span>**

<span style="font-size: 20px;">Computations that reduce multiple elements into a summary value.</span>

| <span style="font-size: 20px;">Function</span>              | <span style="font-size: 20px;">Definition</span>                                      | <span style="font-size: 20px;">Syntax</span>                |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **<span style="font-size: 20px;">`sum()`</span>**           | <span style="font-size: 20px;">Calculates the total of all elements.</span>           | <span style="font-size: 20px;">`torch.sum(input)`</span>    |
| **<span style="font-size: 20px;">`mean()`</span>**          | <span style="font-size: 20px;">Calculates the average (requires float dtype).</span>  | <span style="font-size: 20px;">`torch.mean(input)`</span>   |
| **<span style="font-size: 20px;">`max()` / `min()`</span>** | <span style="font-size: 20px;">Returns the highest or lowest value.</span>            | <span style="font-size: 20px;">`torch.max(input)`</span>    |
| **<span style="font-size: 20px;">`argmax()`</span>**        | <span style="font-size: 20px;">Returns the **index** of the maximum value.</span>     | <span style="font-size: 20px;">`torch.argmax(input)`</span> |
| **<span style="font-size: 20px;">`prod()`</span>**          | <span style="font-size: 20px;">Returns the product of all elements.</span>            | <span style="font-size: 20px;">`torch.prod(input)`</span>   |
| **<span style="font-size: 20px;">`std()` / `var()`</span>** | <span style="font-size: 20px;">Measures data spread (Standard Dev / Variance).</span> | <span style="font-size: 20px;">`torch.std(input)`</span>    |
| **<span style="font-size: 20px;">`all()` / `any()`</span>** | <span style="font-size: 20px;">Logical checks for "Every" or "At least one."</span>   | <span style="font-size: 20px;">`torch.all(input)`</span>    |

***

## **<span style="font-size: 20px;">Critical Learning Notes</span>**

* **<span style="font-size: 20px;">Type Enforcement:</span>**<span style="font-size: 20px;"> PyTorch does not automatically convert types (e.g., `int64` + `float32` will error). You must use `.float()` or `.to()` to match types before operations.</span>
* **<span style="font-size: 20px;">Default Dtypes:</span>**<span style="font-size: 20px;"> Floating-point inputs default to `float32`; integer inputs default to `int64`.</span>
* **<span style="font-size: 20px;">Indexing:</span>**<span style="font-size: 20px;"> Standard Python slicing applies, but remember that `argmax` returns a **flattened index** unless a dimension (`dim`) is specified.</span>
* **<span style="font-size: 20px;">In-place Ops:</span>**<span style="font-size: 20px;"> Functions ending in `_` (like `add_()`) modify the original tensor directly to save memory.</span>
